# dbf-comparator

Инструмент для анализа и сравнения структуры и заполненности двух DBF-файлов:
- **reference.dbf** — эталонный файл
- **test.dbf** — результат парсинга (NiFi / junior-разработчик)

Позволяет быстро выявить расхождения в схеме, пропущенные столбцы и подозрительно низкую заполненность данных.

---

## Возможности

| Функция | Описание |
|---|---|
| Структура схемы | Сравнение набора и порядка столбцов |
| Заполненность | % пустых / заполненных значений по каждому столбцу |
| Join-анализ | Строки только в эталоне / только в тесте по ключевым полям |
| Текстовое резюме | Автоматический вывод аномалий и отклонений |

---

## Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Подготовка файлов

Поместите в корень репозитория:
```
reference.dbf   ← эталон
test.dbf        ← результат NiFi / парсера
```

> **Примечание:** DBF-файлы добавлены в `.gitignore` — не хранятся в репозитории. Файлы нужно добавить вручную в рабочую директорию.

### 3. Запуск ноутбука

```bash
jupyter notebook dbf_comparator.ipynb
```

Или откройте через JupyterLab, VS Code, Google Colab.
